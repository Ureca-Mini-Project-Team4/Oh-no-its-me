## 🌟 주요 기능

| 기능                 | 설명                                                                                             |
| :------------------- | :----------------------------------------------------------------------------------------------- |
| 🗳 **투표 진행**      | 유레카 프론트엔드 대면반 멤버들이 등장하는 밸런스 게임. 4명 중 한 명을 반드시 선택해 투표에 참여 |
| 🏆 **랭킹 시스템**   | 제한 시간 동안 투표를 진행하고, 마감 후 1위를 공개                                               |
| 💬 **익명 커뮤니티** | 로그인 시 랜덤 닉네임이 자동 부여되며, 익명 댓글창에서 자유롭게 소통 가능                        |
